# Ferrel Ganendra

I build small software to understand bigger ideas.

Lately that means AI tools, web apps, and experiments around RAG, agents, and products I can ship. I like the awkward middle of a project: turning a vague idea into something with buttons, errors, data, and users.

From Yogyakarta, Indonesia. Usually coding with coffee nearby.

## field notes

- current rabbit hole: making RAG less fragile
- favorite kind of bug: wrong assumption, not wrong syntax
- default stack: Next.js when UI matters, FastAPI when logic matters
- small belief: shipping teaches faster than polishing private drafts

## working on

- better product taste, mostly by shipping and noticing what feels off
